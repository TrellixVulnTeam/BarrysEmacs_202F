Installing Barry's Emacs

Dependencies

For all platforms you will need:

  • Python 3.4 or newer, 3.5 recommended.
  • pip install xml-preferences
  • pycxx 7.0.0 SVN sources from https://svn.code.sf.net/p/cxx/code/trunk/CXX. Then symlink the checkout folder into Imports/pycxx-7.0.0.

For Linux/xxxBSD:

  • C++ compiler.
  • PyQt5 version 5.6.1 (5.7 should work as well). Install using your platforms package manager.

For Windows:

  • git - https://git-for-windows.github.io/
  • Python 3.5 win64 strongly recommended.
  • Visual C++ 2015 Community Edition (Version 14.0) for Python 3.5 development.
  • py -3.5 -m pip install win-app-packager
  • py -3.5 -m pip install PyQt5
  • inno setup - from http://www.jrsoftware.org/isinfo.php

Note: To create a symlink on windows run the mklink from an admin CMD.

For macOS:

  • XCode
  • XCode command line tools
  • Install Qt 5.6.1
  • Compile and Install PyQt5 from source. pypi version is too hard to package.
  • python2.7 -m pip install dmgbuild
  • python3.5 -m pip install py2app

Build Barry's Emacs

For Linux development:

  1. mkdir -p ~/wc/git
  2. git clone https://github.com/barry-scott/BarrysEmacs.git
  3. cd BarrysEmacs/Builder
  4. . builder_custom_init.sh
  5. ./build-linux.sh
  6. cd ../Editor/PyQtBEmacs
  7. ./bemacs-linux.sh --noredirect

For macOS development:

  1. mkdir -p ~/wc/git
  2. git clone https://github.com/barry-scott/BarrysEmacs.git
  3. cd BarrysEmacs/Builder
  4. . builder_custom_init.sh
  5. ./build-macosx.sh
  6. cd ../Editor/PyQtBEmacs
  7. ./bemacs-macosx.sh --noredirect

For Windows development:

  1. cd %USERPROFILE%
  2. mkdir wc/git
  3. git clone https://github.com/barry-scott/BarrysEmacs.git
  4. cd BarrysEmacs\Builder
  5. builder_custom_init.cmd
  6. "C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\bin\amd64\vcvars64.bat"
  7. build-windows.cmd
  8. cd ..\Editor\PyQtBEmacs
  9. bemacs-win64.sh --noredirect
